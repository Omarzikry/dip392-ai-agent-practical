System: AI Study Assistant Agent

I am building a command-line AI agent that helps students look up study topics using a local knowledge base.
The system uses an agent-based design: the agent validates user input, classifies the intent, selects and
calls the appropriate tool (keyword search or statistics calculator), and returns a structured answer
with recommended next steps. No external AI API is required — the agent relies on local tools and a
Markdown knowledge base. The project is implemented in Python using OOP, type hints, argparse, pathlib,
and pytest for testing.

Planned tools: ValidatorTool, FileReaderTool, SearchTool, CalculatorTool.
Key concepts: Python OOP, type hints, argparse, file I/O, string processing, statistics module, pytest,
modular design, Git version control.
